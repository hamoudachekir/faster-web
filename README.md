## Local Development

### Prerequisites

- PHP 8.2 or higher with extensions: `pdo_mysql`, `pdo_sqlite`, `zip`, `xml`, `mbstring`
- Composer 2.x
- Node.js 18+ and npm
- MySQL (via XAMPP) or SQLite

### Setup

```bash
# Clone the repository
git clone https://github.com/hamoudachekir/faster-web.git
cd faster-web

# Run automated setup (installs dependencies, migrates, seeds database)
composer run setup

# Configure .env file with your database credentials
# For MySQL:
DB_CONNECTION=mysql
DB_DATABASE=faster-web
DB_USERNAME=root
DB_PASSWORD=

# For SQLite (simpler):
DB_CONNECTION=sqlite
DB_DATABASE=/absolute/path/to/database.sqlite

# Start development server (runs server, queue worker, logs, and Vite)
composer run dev
```

### Available Scripts

```bash
composer run setup    # Full setup: dependencies, migrations, seeding, assets
composer run dev      # Start all dev servers (Laravel, queue, logs, Vite)
composer run test     # Run PHPUnit tests
```

   ```env

   APP_DEBUG=false
   ```

### Option 2: Heroku

```bash

heroku addons:create cleardb:ignite  # MySQL database
heroku config:set APP_KEY=$(php artisan key:generate --show)
git push heroku main
heroku run php artisan migrate --seed
```

### Option 3: Traditional Hosting (cPanel)

1. Upload files via FTP/SFTP
2. Point domain to `public/` directory
3. Import database and run migrations
4. Configure `.env` with production credentials
